# Лабораторная работа 1: Система управления учебными курсами


## Описание
Система для управления курсами, преподавателями и студентами университета с поддержкой онлайн и офлайн форматов.



## Функциональность

- Добавление и удаление курсов

- Назначение преподавателей на курсы  

- Управление студентами на курсах

- Поддержка онлайн и офлайн курсов

- Поиск курсов по преподавателю
